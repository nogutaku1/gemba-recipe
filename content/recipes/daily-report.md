## 概要
建設現場の毎日の日報作成は、意外と時間がかかるものです。
疲れて帰ってきた後に、文章を考えるのは大変ですよね。
このレシピでは、ChatGPTを使って、箇条書きのメモからきちんとした日報を作成する方法を紹介します。

## プロンプト
以下のプロンプトをコピーして、ChatGPTに貼り付けてください。
`[ ]` の部分は、その日の状況に合わせて書き換えてください。

```markdown
# 命令書
あなたは建設現場の優秀な現場監督です。
以下の【入力情報】をもとに、会社に提出する「工事日報」を作成してください。
文体は「です・ます」調で、簡潔かつ分かりやすく書いてください。

# 入力情報
- 日付: [今日の日付]
- 現場名: [現場名]
- 天気: [天気]
- 作業内容:
  - [作業1]
  - [作業2]
  - [作業3]
- 人員: [人数]名
- 特記事項: [トラブルや気づきなど]

# 出力フォーマット
【工事日報】
日付：YYYY年MM月DD日
現場名：〇〇建設工事
天気：〇〇

■本日の作業内容
1. 〇〇
2. 〇〇

■特記事項
・〇〇

■明日の予定
・〇〇
```

## 使い方
1. 上記のプロンプトをコピーします。
2. ChatGPT (または Gemini, Claude) を開きます。
3. プロンプトを貼り付け、`[ ]` の部分を自分の現場の情報に書き換えます。
4. 送信ボタンを押すと、日報が生成されます。
